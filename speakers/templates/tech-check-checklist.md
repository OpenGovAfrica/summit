### Tech Check Checklist

Use this for all pre-session tech checks, virtual and in-person. Run through with the speaker 1 to 2 days before their session (virtual) or day-of before doors open (in-person).

### Virtual speakers (live Q&A segment)

**Before the call**
- [ ] Confirm speaker has received platform link
- [ ] Confirm speaker's recorded talk has been received and reviewed
- [ ] Confirm moderator assigned for the session is on the tech check call

**During the call**
- [ ] Camera framing (eye level, good lighting, minimal background distraction)
- [ ] Audio quality (no echo, background noise, or clipping)
- [ ] Internet connection stability check
- [ ] Confirm backup contact method (phone number or alternate email) in case of drop
- [ ] Walk through session flow: recording plays, moderator introduces live Q&A, speaker joins on camera
- [ ] Confirm Q&A format (moderator-relayed vs open floor)
- [ ] Test screen share, if speaker will reference slides during Q&A

**After the call**
- [ ] Log tech check as complete in speakers-by-day file
- [ ] Flag any unresolved issues to production team
### In-person speakers (live session)

**Before doors open**
- [ ] Confirm speaker has arrived and checked in
- [ ] Slides loaded and tested on venue system
- [ ] Backup slides (USB) collected, if provided
- [ ] Microphone and audio levels checked
- [ ] Confirm moderator briefed on speaker's session and Q&A format
- [ ] Confirm livestream feed is capturing audio and video correctly for Virtual Hub

**During walkthrough**
- [ ] Speaker walked through stage layout, timing cues, and Q&A handoff
- [ ] Confirm recording consent has been signed, if not already on file
- [ ] Confirm partnership or sponsor credit is correctly displayed, if applicable

**After the check**
- [ ] Log tech check as complete in speakers-by-day file
- [ ] Flag any unresolved issues to venue AV team
