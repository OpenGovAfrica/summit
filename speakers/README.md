### Speakers

This folder holds everything related to managing speakers for the OpenGov Africa Summit 2026, across all five days and both the Physical Hub (Lagos) and Global Virtual Hub.

### Folder structure

**guides/**
Reusable reference documents on how we run speaker sessions. These rarely change once set.
- `virtual-speaker-guide.md`: format, timeline, and checklist for virtual keynote and headline sessions (pre-recorded talk plus live Q&A)
- `in-person-speaker-guide.md`: format, timeline, and checklist for Physical Hub sessions

**templates/**
Fill-in-the-blank documents copied per speaker, to keep tone and asks consistent across all 60+ sessions.
- `confirmation-email-virtual.md`
- `confirmation-email-in-person.md`
- `speaker-info-request.md`
- `tech-check-checklist.md`

**speakers-by-day/**
One file per summit day, listing confirmed and pending speakers, session format, deadlines, and notes.
- `day1-governance-digital-frontiers.md`
- `day2-climate-mobility.md`
- `day3-health-economic-transformation.md`
- `day4-enterprise-human-capital.md`
- `day5-soft-power-awards.md`

**tracking/**
Status tracking. The Excel workbook (Outreach Tracker tab) remains the source of truth for contact-level detail. Files here, if used, should stay in sync with it rather than fork into a second system.

### Naming convention

- Lowercase, hyphens not underscores
- No dates in filenames unless the file is genuinely versioned by date
- Folder name provides context, filename provides specifics (for example, `guides/virtual-speaker-guide.md`, not `guides/guides-virtual-speaker-guide.md`)
